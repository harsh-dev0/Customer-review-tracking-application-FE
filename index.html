Integrate the Customer Review Widget

To get started with our Customer Review Widget, simply follow these steps:

  1. Add the Script Tag:

    Include the following script tag in the <head> section of your HTML file to load the review widget:

    <script src="https://customer-review-tracking-application-be.onrender.com/review-widget.js"></script>
  2. Customize the Widget:

    You can easily customize the appearance and functionality of the review widget by adding a configuration object in your JavaScript. Hereâ€™s an example:

    window.reviewWidgetConfig = {
    bubbleColor: '#007BFF', // Color of the review bubble
    bubbleTextColor: '#FFFFFF', // Text color inside the bubble
    bubbleSize: '60px', // Size of the review bubble
    popupBackgroundColor: '#FFFFFF', // Background color of the popup
    popupBorderColor: '#ddd', // Border color of the popup
    popupTextColor: '#000000', // Text color in the popup
    reviewHeaderText: 'Customer Reviews', // Header text in the popup
    noReviewsText: 'No reviews available.', // Message when no reviews are present
    submitButtonText: 'Submit Review', // Text on the submit button
    position: { // Position of the bubble
        bottom: '20px',
        right: '20px'
    }
};

Important Note: To view reviews submitted through the widget, registration is required. Make sure users sign up on your platform to access and manage their reviews effectively.
